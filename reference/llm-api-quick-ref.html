Messages 角色对照

角色作用OpenAIAnthropic
system行为规则/人设在 messages 内独立 system 参数
user用户输入messages 内messages 内
assistantAI 回复messages 内messages 内
tool工具返回值messages 内messages 内（content 为 blocks）

OpenAI SDK 速查

from openai import OpenAI
client = OpenAI()  # 读 OPENAI_API_KEY

response = client.chat.completions.create(
    model="gpt-4o-mini",            # 必填
    messages=[...],                   # 必填
    temperature=0.7,                  # 可选，默认 1.0
    max_completion_tokens=1024,       # 可选（推荐，替代 max_tokens）
    top_p=1.0,                        # 可选
)

# 响应访问
response.choices[0].message.content  # str
response.choices[0].finish_reason    # "stop" | "length" | "tool_calls"
response.usage.prompt_tokens         # int
response.usage.completion_tokens     # int

Anthropic SDK 速查

from anthropic import Anthropic
client = Anthropic()  # 读 ANTHROPIC_API_KEY

response = client.messages.create(
    model="claude-sonnet-4-5-20250514",  # 必填
    max_tokens=1024,                       # 必填！
    messages=[...],                        # 必填（第一条必须是 user）
    system="系统提示词",                    # 可选，独立参数
    temperature=0.7,                       # 可选，默认 1.0
    top_p=1.0,                             # 可选
    top_k=40,                              # 可选（Anthropic 专用）
)

# 响应访问
response.content[0].text             # str（注意是 content 数组）
response.stop_reason                 # "end_turn" | "max_tokens"
response.usage.input_tokens          # int
response.usage.output_tokens         # int

参数速查

参数范围推荐值说明
temperature0 - 20（确定性）/ 0.7（创意）越高越随机
max_tokens1 - 模型上限按需Anthropic 必填；OpenAI 可选
top_p0 - 11.0核采样，一般和 temperature 二选一调
top_k1 - ∞40仅 Anthropic，限制候选 token 数
